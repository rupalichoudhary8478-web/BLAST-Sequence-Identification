# BLAST-Sequence-Identification
Identification and functional annotation of unknown DNA and protein sequences using NCBI BLAST and comparative sequence analysis.

Activity 2: Identification of Unknown Biological Sequences Using BLAST
Objective
The objective of this activity is to identify unknown biological sequences using the NCBI BLAST tool and determine:

The type of sequence (DNA or protein)
The organism to which the sequence belongs
The biological significance of the sequence
Interpretation of BLAST output parameters such as percentage identity and E-value
Tool Used
NCBI BLAST (Basic Local Alignment Search Tool)
Website: https://blast.ncbi.nlm.nih.gov/
Methodology
The given unknown sequences were analyzed to determine whether they were nucleotide or protein sequences.
Appropriate BLAST programs were selected:
BLASTn for DNA sequences
BLASTp for protein sequences
Sequences were pasted into the BLAST query box using default parameters.
The top BLAST hits were analyzed based on:
Organism
Percentage identity
Query coverage
E-value
Results were interpreted to identify the biological role of each sequence.
Sequence Analysis and Results

The extremely low E-value and complete identity indicate a highly significant match. The sequence perfectly aligns with the Ebola virus genome.

Biological Function
This sequence forms part of the Ebola virus genome, which encodes proteins essential for viral replication and pathogenicity.

Conclusion
All unknown biological sequences were successfully identified using the NCBI BLAST tool. Sequence 1 was identified as a genomic fragment of Zaire ebolavirus, Sequence 2 as the human insulin gene, and Sequence 3 as the HIV-1 envelope glycoprotein gp160. The results demonstrate the effectiveness of BLAST in identifying and characterizing biological sequences.
